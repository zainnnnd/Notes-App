# 📝 Notes App

A modern Notes Application built using React, Firebase and Tailwind CSS.

## ✨ Features

- 🔐 Firebase Authentication
- 📝 Create Notes
- ✏️ Edit Notes
- 🗑 Delete Notes
- ⭐ Favorite Notes
- 📌 Pin Notes
- 🔍 Search Notes
- 🚪 Logout
- 🎨 Modern UI
- ⚡ Framer Motion Animations

## 🛠 Tech Stack

- React
- Firebase Authentication
- Cloud Firestore
- Tailwind CSS
- React Hot Toast
- Framer Motion
- React Router
